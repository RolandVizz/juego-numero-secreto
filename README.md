# Juego del número secreto
Si deseas probar la página y su funcionamiento, puedes hacerlo dando click [aquí](https://rolandvizz.github.io/juego-numero-secreto/).

Este es el clásico juego del "número secreto", donde deberas adivinar el número que la página genere.\
A continuación enlisto las reglas:
- No tiene límite de intentos
- El número generado es del 1 al 20
- Escribir solo números

En el momento que adivines el número, se desbloqueara el botón de nuevo juego. Con este podrás reiniciar la aplicación y continuar jugando.

**Esta aplicación fue desarrollada como un ejercicio dentro del programa Oracle One impartido por Alura.**\
